# Yui

> Note: this project is still in development!

Bot I use for messenger group chats


### How to run

Clone this repo `Yui`

```sh
python3.11 -m pip install git+https://github.com/jmmaa/yui.git
```

then you can run

```sh
python3.11 -m yui
```
