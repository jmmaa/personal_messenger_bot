# Yui

> Note: this project is still in development!

Bot I use for messenger group chats


### How to run

Clone this repo `Yui`

```sh
git clone https://github.com/jmmaa/yui.git
```

then run the build script `scripts/build`

```sh
scripts/build
```

then `scripts/run`
```sh
scripts/run
```
